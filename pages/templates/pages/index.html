As of now, there is no evidence whatsoever and very little feasibility that Collegeboard could track student searches. There are a myriad of other sites that students could better cheat on. This is simply an educational site designed to prepare students for their exams. If you want to download these notes or print them before test day, that's your choice! This site is planned to run throughout and during this years' exams.

Go to a course and see if you can create a page there or in one of its units. If you can't, it's likely because the page was locked, or you don't have specific permisissions needed. If you want to edit a locked page, you can report it, and our team will consider reopening it up for access.

The site uses a Markdown based editor. If you have trouble using it, you can read the editor guide.

If you see notes that are misleading, trollish, or just plain wrong, please report it! Each report notifies our team immediately and is then carefully looked at. Thank you for keeping AP Crowd the resource that it is!

Furthermore, each note/page has one of 4 tiers of protection: none, confirmed users only, locked, and locked with no children.

  • None: All users can edit
  • Confimred users only: Only users that have had an account for more than five days are able to edit
  • Locked: Only the user that locked the note can edit
  • Locked with no children: Only the user that added this protection can edit the note, and add sub-pages

Protections can be added by users or staff.

I'm sure the site has loads of room for improvement. Bugs can be emailed to us (apcrowd2020 at gmail dot com) or listed under the "Issues" section of our Github. Suggestions are also welcome in the inbox or in our Discord. I want this to be the best resource that it can be, so if you have any input, I'd love to hear it!

First and foremost, please add your notes! If you want to help in other ways, you can contribute to our Github or help on our Discord. The site runs on Django (a Python framework for web applications) and through sheer force of will.
